# otus-cpp-basics-hw03
COMMAND_LINE:

`-table`
Print of a score table and exit from the program
```sh
./otus-cpp-basics-hw03 -table
```

`-max [value > 0]`
Set the maximum spread level
```sh
./otus-cpp-basics-hw03 -max 35
```

`-level [1-3]`
Set the game level
```sh
./otus-cpp-basics-hw03 -level 2
```
